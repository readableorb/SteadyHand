# SteadyHand

## Features:
An Assistive Stabilizing Spoon that counteracts hand tremors in real time.

Custom pcb that is small and powered by a 6V battery.

3D printed case

Raspberry Pi Pico

9G Micro Servo Motors (2x)

Switch

Battery Slot

Adjustable Buck Boost Converter

## BOM:
This is what you need to make this hackpad.

- MPU-6050
- Raspberry Pi Pico
- 9G Micro Servo Motors (2x)
- Buck Boost Converter
- Switch
- 470 MicroFarad Capacitor (2x)
- 1000 MicroFarad Capacitor
- 330 PicoFarad Ceramic Capacitor
